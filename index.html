Virtual Accounts (VA) · GL / CoA
Виртуальный счёт для каждого ТСП — учёт рублёвых поступлений и обязательств по выплатам. Поверх VA — бухгалтерский слой: проводки (GL) по каждой операции, план счетов (CoA), отчётность P&L, баланс, ОСВ.
Терминалы (MID) в НКО «Твои Платежи»
MID-001 MCC 5999 · Маркетплейс
MID-002 MCC 7372 · SaaS / Digital
MID-003 MCC 4722 · Туризм
MID-004 MCC 8299 · Образование
MID-N Per TSP / Per MCC

🧅
Прозрачность для ТСП
ТСП видит только NBPay API. НКО «Твои Платежи» — невидимый rail под капотом. Смена провайдера без изменений на стороне ТСП.
🎛️
Терминал = MID
Виртуальный мерчант-идентификатор в НКО «Твои Платежи». Один терминал на MCC-сегмент (или на каждый ТСП). Покупатель разницы не замечает.
💱
FX Settlement
NBPay принимает RUB, конвертирует по рыночному курсу с маржой и переводит ТСП в USD / EUR / AED на указанные реквизиты.
🪪
Онбординг ТСП
KYB/KYC на стороне NBPay. НКО «Твои Платежи» онбордит NBPay как единого мастер-мерчанта-агрегатора. Пакет документов по каждому ТСП — в НКО «Твои Платежи».
