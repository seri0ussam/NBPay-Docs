🏦 Аналитические счета (VA) · GL
Для каждого ТСП открывается виртуальный аналитический счёт — бухгалтерский инструмент для учёта баланса, начислений (за вычетом комиссий) и выплат. Реального движения денег нет — это только учёт. Поверх VA — бухгалтерский GL: проводки по каждой операции, P&L, баланс.
Терминалы субмерчантов в НКО «Твои Платежи»
TSP-001 merchantCode · MCC 5999
TSP-002 merchantCode · MCC 7372
TSP-003 merchantCode · MCC 4722
TSP-N один субмерчант = один терминал
🔒
PCI DSS: карточные данные вводятся на странице НКО «Твои Платежи» и не проходят через NBPay. Периметр PCI — на стороне ТП (L1). NBPay заполняет SAQ-A. Если NBPay начнёт принимать карточные данные напрямую — потребуется собственная сертификация.

🧅
Прозрачность для ТСП
ТСП видит только NBPay API и ЛК. НКО «Твои Платежи» — невидимый rail. Смена или добавление провайдера без изменений на стороне ТСП.
🎛️
Терминал = субмерчант в ТП
Каждый ТСП — отдельная учётная запись в ТП с уникальным merchantCode и MCC. Изоляция лимитов и рисков. Покупатель разницы не замечает.
💱
FX Settlement
NBPay принимает RUB от ТП, конвертирует по рыночному курсу с маржой и переводит ТСП в USD / EUR / AED на зарубежные реквизиты.
🪪
Двухэтапный онбординг
KYB в NBPay + регистрация субмерчанта в ТП. Скорость активации ТСП зависит от скорости проверки в ТП — ключевой операционный параметр.
